# Drupal Citrus Project

This is a Drupal9 eShop project made for [Citrus Solutions](www.citrus.fi) as part of the Full-stack Web Development program from [Helsinki Business College](www.bc.fi)

## Technologies used

Built with:

- Drupal 9
- Composer
- Commerce Core
- Lando
- Docker
- SCSS/SASS
- Javascript
- TWIG
